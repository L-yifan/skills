# React Path Reference

Use this path when the user requests React or when shadcn/ui components are needed.

## Initialization

```bash
bash scripts/init-artifact.sh <project-name>
# or
bash scripts/init-artifact.sh <project-name> --framework react
```

## Includes

- React 18 + TypeScript + Vite
- Tailwind CSS 3.4.1
- shadcn/ui component set and Radix dependencies
- Path alias `@/`

## Good Fit

- Dashboard-style apps with reusable component primitives
- Complex forms and interactions needing shadcn/ui building blocks
- Multi-view UIs that benefit from React ecosystem conventions
