## Skills CLI 使用方法

### 安装技能

```bash
# 安装到用户级别（推荐）
npx skills add <https://github.com/owner/repo --skill skill> -g

# 安装到项目级别
npx skills add <https://github.com/owner/repo --skill skill>
```

### 管理已安装的技能

```bash
# 列出所有已安装的技能
npx skills list -g

# 检查更新
npx skills check -g

# 更新所有技能
npx skills update -g

# 移除技能
npx skills remove <skill-name> -g

# 搜索新技能
npx skills find [query]
```
